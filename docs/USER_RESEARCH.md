# User Research Plan (5 Interviews)

Goal: validate whether this tool is intuitive and whether the commercial-aware surfaces are understandable and useful.

## Participants
1. “Viewer mindset” (non-technical): 2 participants
2. “Product/analytics mindset”: 2 participants
3. “Ads/monetization mindset”: 1 participant

## Method
- 30 minutes moderated remote session
- Tasks + think-aloud + short debrief

## Tasks
1. Search for a “vibe” query and pick something to watch.
2. Use filters to narrow options.
3. Explain what the “Ranking Tradeoff” slider does.
4. Interpret the Monetization tab: advertiser fit + brand risk.
5. (If enabled) react to AI Hook + Ad Strategy. Do you trust it? Why/why not?

## Questions
- What do you expect from “Smart-Scout”? What surprised you?
- What signals would make you trust the recommendations?
- What would you remove to make this faster to use?
- What would you add if you were using this daily?

## Output
- Top 5 usability issues
- Proposed UI changes
- Clear “keep/change/kill” decision on:
  - slider
  - AI insights
  - brand-safety display
